# Frontend

React + TypeScript + Vite frontend for AI Smart Task Manager.

## Run locally

```bash
npm install
npm run dev
```

Default URL: `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```

## Important

- API base URL is defined in `src/services/api.ts`.
- Backend should run at `http://localhost:5001` for local development.

For full-stack setup and deployment details, see the root `README.md`.
